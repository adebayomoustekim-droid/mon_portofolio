Gestion des employés

Application web pour gérer les employés par département avec authentification sécurisée.

Description du projet

Ce projet est une application web complète destinée à la gestion des employés d'une entreprise. Il permet de créer, modifier et supprimer des employés, assigner des départements, suivre leurs performances et générer des rapports automatisés. L'objectif principal était de créer une interface intuitive tout en assurant la sécurité et la rapidité du système.

Technologies utilisées

  • PHP
  • MySQL
  • Bootstrap
  • HTML / CSS / JS
